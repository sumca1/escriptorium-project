## 3 רמות Augmentation

### 🟢 Light
```python
OCRAugmentor(level='light')
```
- סיבוב ±2°
- רעש מינימלי
- מתאים לדאטא נקי

### 🟡 Medium ⭐ ברירת מחדל
```python
OCRAugmentor(level='medium')
```
- סיבוב ±5°
- רעש + טשטוש + מורפולוגיה
- מתאים לרוב המקרים

### 🔴 Heavy
```python
OCRAugmentor(level='heavy')
```
- סיבוב ±10°
- כל הטרנספורמציות
- מתאים לדאטא רועש מאוד
