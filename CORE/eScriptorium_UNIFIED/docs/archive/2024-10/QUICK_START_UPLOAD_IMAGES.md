# 📸 מדריך מהיר - העלאת תמונות ל-eScriptorium

**תאריך**: 19 אוקטובר 2025

---

## 🎯 תהליך העלאת תמונות - צעד אחר צעד

### שלב 1️⃣: יצירת פרויקט (Project)

1. היכנס ל-**http://localhost:8082**
2. התחבר עם שם המשתמש והסיסמה שלך
3. בדף הראשי, לחץ על **"Create Project"** או **"צור פרויקט"**
4. תן שם לפרויקט (למשל: "ספר עברי 1")
5. לחץ **"Create"** / **"צור"**

### שלב 2️⃣: יצירת מסמך (Document)

1. לאחר יצירת הפרויקט, היכנס לפרויקט (לחץ על השם)
2. בתוך הפרויקט, לחץ **"Create Document"** או **"צור מסמך"**
3. תן שם למסמך (למשל: "עמוד 1-10")
4. (אופציונלי) בחר שפה ראשית
5. לחץ **"Create"** / **"צור"**

### שלב 3️⃣: העלאת תמונות

עכשיו יש לך **שתי אפשרויות**:

#### אפשרות א': העלאה ידנית דרך הממשק

1. היכנס למסמך שיצרת
2. לחץ על הכפתור **"Images"** בתפריט העליון
3. בעמוד Images, תראה כפתור **"Import"** או **"Upload Images"**
4. לחץ על הכפתור ובחר תמונות מהמחשב שלך
5. התמונות יעלו ויופיעו ברשימה

#### אפשרות ב': העלאה באמצעות Import Wizard

1. מתפריט המסמך, בחר **"Import"**
2. בחר **"Images from disk"** / **"תמונות מהדיסק"**
3. גרור ושחרר קבצים או לחץ לבחירה
4. פורמטים נתמכים: JPG, PNG, TIFF, PDF
5. לחץ **"Upload"** / **"העלה"**

```
📁 Project (פרויקט)

        └── 🖼️ Images (תמונות)

             └── Image 3
```

**חשוב**: אי אפשר להעלות תמונות ישירות לפרויקט - **חייבים ליצור מסמך קודם**!

## 🎨 שימוש ב-Image Enhancement (שיפור תמונה)

לאחר העלאת התמונות:

1. עבור לעמוד **Images** של המסמך
2. ליד כל תמונה תראה כפתור **🎨 "Enhance"**
3. לחץ עליו לשיפור איכות התמונה
4. כוונן פרמטרים:
   - **Sharpen** (חידוד): 0-2.0
   - **Denoise** (הפחתת רעש): 0-1.0
   - **Contrast** (ניגודיות): 0.5-2.0
   - **Brightness** (בהירות): 0.5-2.0
5. לחץ **"Process"** לעיבוד

## 🔍 פתרון בעיות נפוצות

### ❌ "לא רואה כפתור ליצירת פרויקט"

**פתרון**: ודא שאתה מחובר למערכת. אם אין לך משתמש:

```powershell
docker-compose exec web python manage.py createsuperuser
```

עקוב אחר ההנחיות וצור משתמש.

---

### ❌ "הכפתור Create Document אפור/מנוטרל"

**פתרון**: ודא שאתה **בתוך הפרויקט** ולא בדף הראשי של הפרויקטים.

---

### ❌ "התמונות לא עולות"

**פתרונות**:

1. **בדוק גודל הקובץ**: מגבלת ברירת המחדל היא 150MB
2. **בדוק פורמט**: רק JPG, PNG, TIFF, PDF נתמכים
3. **בדוק לוגים**:
   ```powershell
   docker-compose logs -f web
   ```

---

### ❌ "אין אפשרות Enhance"

**פתרון**: ודא ש-FastAPI רץ:

```powershell
docker-compose ps fastapi
```

אם לא רץ:

```powershell
docker-compose restart fastapi
```

## 📊 סטטוס המערכת

### בדיקה מהירה:

```powershell
# בדוק שכל השירותים רצים
docker-compose ps

# ודא ש-FastAPI עובד
curl http://localhost:8001/health
```

```json
{"status": "healthy", "version": "1.0.0"}
```

## 🆘 עזרה נוספת

אם עדיין יש בעיות:

1. **לוגים של web**:
   ```powershell
   docker-compose logs -f web
   ```

2. **לוגים של FastAPI**:
   ```powershell
   docker-compose logs -f fastapi
   ```

3. **אתחול המערכת**:
   ```powershell
   docker-compose restart web nginx fastapi
   ```

4. **אתחול מלא**:
   ```powershell
   docker-compose down
   docker-compose up -d
   ```

## 🎯 תזרים עבודה מומלץ

### לעבודה על ספר/כתב יד:

1. **צור פרויקט** אחד לכל ספר
2. **צור מסמך** לכל יחידה (פרק, עמודים 1-50, וכו')
3. **העלה תמונות** למסמך
4. **שפר תמונות** באמצעות Enhance (אופציונלי)
5. **הפעל Segment** לזיהוי שורות
6. **הפעל Transcribe** לתמלול
