## 🔍 שלב 5: בדיקות מודלים (Modals)

### מודל חפש (Search Modal)
1. לחץ על כפתור "Search" / "חיפוש"
2. בדוק את הטקסט:
   - [ ] כותרת: `חפש טקסט ב`
   - [ ] Label: `חפוש`
   - [ ] כפתור: `חיפוש`

### מודל הוסף קבוצה (Add Group Modal)
1. פתח עמוד שיתוף (Share page)
2. בדוק את הטקסט:
   - [ ] כותרת: `הוסף קבוצה או משתמש`
   - [ ] תת-כותרת: `הוסף קבוצה`
   - [ ] תת-כותרת: `הוסף משתמש`
   - [ ] כפתור: `הוסף`

### מודל מחיקה
1. לחץ על "Delete" בכל דוגמא
2. בדוק את הטקסט:
   - [ ] כותרת: `בטול`
   - [ ] כפתור: `מחוק`
   - [ ] כפתור: `ביטול`

## 🐛 טיפול בבעיות

### אם טקסט לא תורגם
1. בדוק ב `he.json` אם יש מפתח
2. בדוק ב Vue component אם משתמש `$t('key')`
3. בדוק ב Django settings אם עברית מוגדרת
4. בדוק ב Browser console (F12) לשגיאות

### אם הדף לא טועמן
1. בדוק את הקונסול של Django
2. בדוק בעמוד Browser errors (F12)
3. בדוק אם השרתון עדיין פעול

### אם הטקסט בלבול
1. בדוק RTL settings
2. בדוק CSS direction
3. בדוק HTML dir attribute
