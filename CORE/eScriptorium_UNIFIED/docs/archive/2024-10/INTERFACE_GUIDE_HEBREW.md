# מדריך מפורט עם תרגום ממשק לעברית

## 📌 מילון מונחים - אנגלית ↔ עברית

| אנגלית | עברית | היכן מופיע |
|--------|-------|------------|
| **Projects** | פרויקטים | תפריט עליון |
| **Create Project** | יצירת פרויקט | כפתור בדף פרויקטים |
| **Documents** | מסמכים | בתוך פרויקט |
| **Create Document** | יצירת מסמך | כפתור בתוך פרויקט |
| **Images** | תמונות | טאב בתצוגת מסמך |
| **Import** | ייבוא | כפתור להעלאת קבצים |
| **Edit** | עריכה | כפתור עפרון 🖊️ |
| **Segmentation** | פילוח | תפריט עליון |
| **Train Model** | אימון מודל | תת-תפריט של פילוח |
| **Segment** | פלח / בצע פילוח | כפתור בעריכה |
| **Models** | מודלים | תפריט עליון |
| **Transcription** | תמלול / תעתיק | טאב בעריכה |
