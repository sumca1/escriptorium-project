## 🚀 המערכת מוכנה לשימוש מלא!

### תהליך עבודה זמין עכשיו:

#### שלב 1: יצירת קורפוס GT
```bash
1. גשו ל: http://localhost:8082/admin/
2. בחרו: TABA_PIPELINE → Ground Truth Corpora → Add
3. מלאו: שם, תיאור, סוג מקור = "Text Files"
4. נתיב מקור: /opt/taba_pipeline/data/ground_truth/hebrew_classics/
```

#### שלב 2: הוספת טקסטים לקורפוס  
```bash
1. בחרו: TABA_PIPELINE → Ground Truth Texts → Add
2. בחרו קורפוס קיים
3. הוסיפו כותרת ותוכן טקסט נקי
4. שמרו
```

#### שלב 3: יצירת עבודת יישור
```bash
1. גשו ל: http://localhost:8082/taba/
2. "יצירת עבודה חדשה"
3. בחרו מסמך עם OCR גולמי
4. בחרו קורפוס GT מתאים
5. הגדירו פרמטרי Passim (ברירת מחדל מתאימה)
6. "הרץ יישור"
```

#### שלב 4: קבלת תוצאות
```bash

✓ קובץ מוכן לאימון מודלים חדשים
```

## 💡 טיפים לשימוש מיטבי

### להתחלה:
1. **התחילו עם טקסט פשוט** - בחרו מסמך עם OCR סביר (>60% דיוק)
2. **השתמשו בGT רלוונטי** - וודאו שהקורפוס תואם לתוכן המסמך  
3. **התחילו עם פרמטרים בסיסיים** - N-gram=7, cores=6, memory=8GB
4. **בדקו תוצאות ידנית** - ודאו שהשיפור אכן משמעותי

### לשיפור ביצועים:
1. **הוסיפו טקסטים נוספים לקורפוס** - יותר GT = יותר דיוק
2. **התאימו פרמטרי Passim** - לטקסטים מורכבים: N-gram=5-9
3. **השתמשו בסינון** - רק תוצאות עם דיוק >80%
4. **אמנו מודלים חדשים** - השתמשו בתוצאות לאימון Kraken
