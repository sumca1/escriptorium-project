# תיקיית תרגומים נקייה

תיקייה זו מיועדת לתרגומים עתידיים.

## סטטוס נוכחי: נקי לחלוטין ✅
- אין תרגומים ישנים מזוהמים
- מוכן לתרגומים חדשים
- אין קבצי django.po עבריים מהמערכת הישנה

## להוספת תרגום עברי בעתיד:
```bash
python manage.py makemessages -l he
# ערוך קבצי .po
python manage.py compilemessages
```

---
**חשוב:** אין להעתיק קבצי תרגום מהמערכת הישנה!
