## 🚀 הפעלה מהירה

### אופציה 1: סקריפט אוטומטי (מומלץ)

```powershell
cd escriptorium\ui\control-center
.\START.ps1
```

**מה זה עושה:**
- ✅ מעתיק קבצי תיעוד אוטומטית
- ✅ מפעיל שרת HTTP
- ✅ פותח את הדשבורד בדפדפן

### אופציה 2: ידני

```powershell
cd escriptorium\ui\control-center

# העתק קבצי תיעוד
Copy-Item ..\..\..\SESSION_LOG.md .
Copy-Item ..\..\..\CURRENT_STATE.md .

# הפעל שרת
python -m http.server 8080

# פתח בדפדפן
start http://localhost:8080/dashboard.html
```

## 🔧 איך זה עובד?

### הארכיטקטורה:

```
1. לחיצה על כפתור "SESSION_LOG.md"
         ↓
2. dashboard.html מפעיל viewDocumentation()
         ↓
3. טעינת markdown-converter.js
         ↓
4. קריאת קובץ ה-.md מהשרת
         ↓
5. המרה ל-HTML מעוצב
         ↓
6. זיהוי עברית + הוספת dir="rtl"
         ↓
7. פתיחה בחלון חדש ✨
```

### הקבצים:

| קובץ | תפקיד |
|------|-------|
| `markdown-converter.js` | המרת MD → HTML |
| `dashboard.html` | פונקציית `viewDocumentation()` |
| `overview.js` | כפתורים לתיעוד |
| `START.ps1` | הפעלה אוטומטית |

## 🎯 תכונות מתקדמות

### 1. זיהוי אוטומטי של עברית

```javascript
// הקוד מזהה תווים עבריים/ערביים
const rtlPattern = /[\u0590-\u05FF\u0600-\u06FF]/;

// ומוסיף dir="rtl" אוטומטית
if (rtlPattern.test(content)) {
    return `<p dir="rtl">${content}</p>`;
}
```

### 2. המרת טבלאות Markdown

```markdown
| כותרת 1 | כותרת 2 |
|---------|---------|
| ערך 1   | ערך 2   |
```

הופך ל:
```html
<table class="markdown-table">
    <thead>...</thead>
    <tbody>...</tbody>
</table>
```

### 3. קוד עם Syntax Highlighting

```markdown
```javascript
const x = 5;
```
```

הופך לבלוק קוד מעוצב עם רקע כהה.

## 🐛 פתרון בעיות

### בעיה: "Failed to fetch"
**פתרון:** ודא שהשרת HTTP רץ:
```powershell
python -m http.server 8080
```

### בעיה: התיעוד לא מעודכן
**פתרון:** העתק שוב את הקבצים:
```powershell
.\START.ps1
```

### בעיה: העברית לא RTL
**פתרון:** הממיר מזהה אוטומטית - בדוק שהקובץ מכיל עברית.
